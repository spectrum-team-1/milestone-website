Group 15 - Spectrum Plays Spring 2024

Project Overview:

For the Kennesaw Spring 2024 IT Capstone, our team was tasked to build a web app for Spectrum Plays. A revolutionary new way to teach sheet music to children grades K-5.

Using color notation to show the learner what notes represent what key, we can break through the wall that most children run into when learning something as complicated as sheet music.

We used industry standard technologies like React, C#, and Typescript to build this web application with the ideas of scalability, compatibility, and extensibility.
